Features
--------

    * Data structures
        * **Arrays**
        * **Linked Lists**
        * **Stacks**
        * **Queues**
        * **Maps & Hash Tables**
        * **Graphs**
        * **Trees**
        * **Binary Trees & Binary Search Trees**
        * **Self-balancing Trees (AVL Trees, Red-Black Trees, Splay Trees)**
        * **Heaps**
        * **Disjoint Set Union**
        * **Minimum Spanning Trees**
    * Sorting
        * **Bubblesort**
        * **Mergesort**
        * **Heapsort**
        * **Insertionsort**
        * **Quicksort**
        * **Selection sort**
        * **Pigeonhole sort.**
        * **Binary tree sort.**
    * Searching
        * **Linear search**
        * **Binary search**
    * Graph based DSA
        * **Graph Data Structure**
        * **Spanning Tree**
        * **Strongly Connected Components**
        * **Adjacency Matrix**
        * **Adjacency List**
        * **DFS Algorithm**
        * **Breadth-first Search**
        * **Bellman Ford's Algorithm**
    * Greedy Algorithms
        * **Greedy Algorithm**
        * **Ford-Fulkerson Algorithm**
        * **Dijkstra's Algorithm**
        * **Kruskal's Algorithm**
        * **Prim's Algorithm**
        * **Huffman Code**
    * Dynamic Programming
        * **Backtracking Algorithm**
        * **Rabin-Karp Algorithm**
